# Tools

My repository of all bug bounty hunting tools

## Tool Descriptions
Tool | Description
---- | ----
Recon API | A simple API which returns useful recon data on port 6301
Backup | Basic backup script which reads from a JSON configuration file
Install.sh | Initializes a new hunting box. Made to run once on a Digital Ocean, AWS, or Raspberry Pi box (or any linux box)
ReconParse | Ruby script which parses a file for a number of different formats
